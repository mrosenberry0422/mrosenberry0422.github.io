UCF Mad Libs

This was my final project in the course, Concepts in Computer Science(2500C), which focused on the basics of Python. I made a progam that created a UCF themed mad libs with user input. Using if-statemts, loops, and fuctions, I asked the user to input multiple nouns, verbs, and adjectives, just like a normal mad libs.
